<div class="_bullet" v-click="[0]">

- `typescript-eslint`では、`ESLintUtils.RuleCreator` を使用してルールを作成

```ts
import { ESLintUtils } from "@typescript-eslint/utils";

export const sampleRule = ESLintUtils.RuleCreator.withoutDocs({
  meta: {
    // ルールのメタデータを記述
  },
  create(context) {
    // ルールを記述
  },
});
```

</div>
